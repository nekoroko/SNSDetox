SNS Detox Icon Files
=================

This directory should contain the following icon files:
- icon128.png (128x128 pixels)

To generate these icons:

1. Open the generate_icons.html file in your browser
2. For each icon size, click the "Download" button
3. Save the downloaded files to this directory with the exact names listed above

The icons are required for the Chrome extension to work properly. They will be displayed in various places:
- Browser toolbar
- Extensions page
- Chrome Web Store (if published)
- Installation dialog

If you prefer to create your own icons, make sure they are in PNG format and have the correct dimensions.
